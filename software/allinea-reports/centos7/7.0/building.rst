

Download and extract::

  $ cd /afs/pdc.kth.se/pdc/vol/allinea-reports/src/
  $ wget http://content.allinea.com/downloads/allinea-reports-7.0-Redhat-7.0-x86_64.tar
  $ tar xvf allinea-reports-7.0-Redhat-7.0-x86_64.tar
  $ cd allinea-reports-7.0-Redhat-7.0-x86_64

Then run the text installer::

  $ ./textinstall.sh

As install base directory select::

 /afs/pdc.kth.se/pdc/vol/allinea-reports/7.0

  /afs/pdc.kth.se/pdc/vol/allinea-reports/licences/License
