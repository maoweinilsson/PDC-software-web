The AVL/Fire files can be accessed by loading the appropriate modules. To see which versions of AVL/Fire are available use the command

.. code-block:: bash

 module avail avl

In order to use AVL/Fire at PDC you must provide your own license for the software.  You can  use the environment variable

.. code-block:: bash

 CDLMD_LICENSE_FILE

to set to point to the correct license server.


Formation on how to submit jobs on Lindgren, see `Job Submission on Lindgren <https://www.pdc.kth.se/resources/computers/lindgren/how-to/run>`_ .
